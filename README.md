# Overview
krab is a shell designed to be lightweight and minimal while not relying on any external libraries (like GNU Readline or Libedit).
# Installition
you can simply install it with ```make```
```
make install
```
# Uninstallition
"why?" why not make life a little easier?
```
make uninstall
```
# Dependencies
There is none.
